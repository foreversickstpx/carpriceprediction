# Car Price Prediction — регрессионная ANN на NumPy

Искусственная нейронная сеть для предсказания цены подержанного автомобиля, реализованная **с нуля на NumPy** (без использования PyTorch/TensorFlow) — с полным циклом forward propagation и backpropagation, написанным вручную.

## Описание проекта

Проект демонстрирует понимание внутренней механики нейронных сетей: матричные операции, инициализация весов, функции активации, градиентный спуск и backpropagation реализованы напрямую через NumPy, без использования готовых ML-фреймворков для самой модели.

Задача — **регрессия**: предсказание цены автомобиля (USD) по пяти числовым признакам.

## Датасет

Синтетический датасет из 500 примеров, сгенерированный с реалистичной нелинейной зависимостью (экспоненциальная амортизация по возрасту, влияние пробега, мощности, объёма двигателя и количества владельцев + рыночный шум).

**Признаки (X):**
| Признак | Описание |
|---|---|
| `year` | Год выпуска |
| `mileage_km` | Пробег, км |
| `engine_volume` | Объём двигателя, л |
| `horsepower` | Мощность, л.с. |
| `owners_count` | Количество предыдущих владельцев |

**Целевая переменная (y):** `price_usd` — цена в долларах

## Архитектура сети

```
Input(5) → Hidden(16, ReLU) → Hidden(8, ReLU) → Output(1, linear)
```

- **He-инициализация** весов — для устойчивого обучения со слоями ReLU
- **ReLU** в скрытых слоях для нелинейности
- **Без активации на выходе** — корректно для регрессии, где целевая переменная не ограничена диапазоном [0, 1]
- **MSE (Mean Squared Error)** в качестве функции потерь
- **Mini-batch / full-batch градиентный спуск** для обновления весов

## Стек технологий

- Python 3
- NumPy — вся математика сети (forward/backward pass) реализована вручную
- pandas — загрузка и обработка табличных данных
- scikit-learn — `train_test_split`, `StandardScaler` (только препроцессинг, не модель)
- Matplotlib — визуализация кривой обучения и качества предсказаний

## Препроцессинг данных

- Разделение train/test (80/20)
- Стандартизация признаков и целевой переменной (z-score), обученная только на train, применённая к test без повторного fit — без утечки данных (data leakage)

## Результаты

Модель достигает стабильного снижения MSE на тренировочной выборке и адекватного качества предсказаний на тестовой выборке (см. график "предсказание vs реальность" в notebook) — без признаков переобучения при текущей архитектуре и размере датасета.
